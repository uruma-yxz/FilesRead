### Este projeto foi desenvolvido com o objetivo de praticar a manipulação de arquivos e os conceitos de Programação Orientada a Objetos (POO) na linguagem C++.

# 🎯 Objetivos de Estudo

    Biblioteca <fstream>: Exploração das classes ifstream, ofstream e fstream para persistência de dados.

    POO (Programação Orientada a Objetos): Aplicação de conceitos como classes, encapsulamento e modularização de código.

    Arquitetura de Pastas: Organização de um projeto C++ em múltiplos arquivos e diretórios.

# 🏗️ Estrutura do Projeto

Baseado na organização atual:

    src/Home: Controla o fluxo principal e a interface com o usuário.

    src/Input: Gerencia a captura de dados (como o nome dos arquivos).

    src/useCase: Contém a lógica de negócio para leitura e escrita de arquivos.
